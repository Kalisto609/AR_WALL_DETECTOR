## Setup Steps
- Unity 6000.2.12f1
- AR Foundation + ARCore XR Plugin
- Minimum Android API 25 (Android 7.1 Nougat)+
- Graphics API: OpenGLES3 (Vulkan disabled)
- Build Target: Android

## Addressables Config
- Group: RemoteVideos
- earthquake_video → Video 1
- video_2 → Video 2

## CDN Details
- Hosted on Netlify (CDN)
- URL: https://lucent-manatee-75c3ea.netlify.app/Android

## How To Test
- Build to Android device
- Point camera at wall
- Tap detected plane
- Use buttons to switch videos
